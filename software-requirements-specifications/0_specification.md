スマートフォン，PC，タブレットのいずれでもプレイ可能な，HTMLベースのランキング機能付き落ち物パズルゲームを作成します．
画面の構成:は縦長（スマートフォンサイズを基準とする）のゲームステージとします．
ステージ内には，丸い形状のキャラクター（以下、オブジェクト）が画面上部から多数降り注ぎ，重力に従ってランダムに積み重なります．オブジェクト同士は円形の当たり判定を持ち，隙間に滑り込んだり，崩れたりする物理挙動が必要です．5〜6種類程度の異なるデザインのオブジェクトがランダムに登場します．
なぞってつなげることで消去することができます．たとえば，プレイヤーが画面をドラッグ（またはスワイプ）し，同じ種類のオブジェクトを隣り合っているもの同士で3つ以上，一筆書きのようになぞって繋げると，それらのオブジェクトが選択状態になります．指やマウスを離した瞬間に，選択されたオブジェクトが消滅（スコア加算）します．オブジェクトが消えると，上にあったオブジェクトが重力で下に落ち，空いた上部のスペースに新しいオブジェクトが自動的に補充されます．
1ゲームの制限時間は60秒，残り時間が0になるとゲーム終了．
HTML5, CSS3, JavaScript（Canvas API または Canvasベースのゲームライブラリ/物理エンジン（例: Matter.js, Phaser 3等）の使用を想定しています．PC，スマートフォン，タブレットのどの画面サイズでも，ゲームのプレイ領域（アスペクト比）が適切に維持され，中央に配置されること．
ゲーム終了時，獲得したスコアとユーザー識別子（ユーザー名など）を非同期通信（API）でサーバーに送信し，データベースに保存します．
ゲーム起動時，または終了時にサーバーから全ユーザーのランキング上位10人を取得して画面に表示します．
